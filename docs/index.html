⚠️ macOS Installation Note

This app is not yet notarized by Apple. After downloading, you'll need to run this command in Terminal to remove the quarantine flag:

xattr -d com.apple.quarantine '/Applications/Databricks Deployer.app'

Why? macOS Gatekeeper blocks apps that aren't code-signed and notarized by Apple. This command tells macOS to trust the app.
